Os Principais Comandos para se criar um site

Estrutura HTML Básica

HTML (HyperText Markup Language):Primeiro começamos com o html que é a estrutura mais basica para se criar um site, o html basicamente é a linguagem padrão para criar páginas da web. Ela usa uma estrutura de tags para organizar e exibir o conteúdo. Algumas das principais tags incluem <html>, <head>, <body>, <h1> a <h6> (para títulos), <p> (para parágrafos), entre outras.

Estilizando com CSS

CSS (Cascading Style Sheets): Em seguida do html vem ele, o css, que é a linguagem usada para definir a apresentação e o estilo de uma página web. Com ele, podemos alterar cores, fontes, espaçamento e muito mais. Por exemplo, o comando `color: blue;` muda a cor do texto para azul.
